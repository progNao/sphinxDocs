3. Reactの開発環境構築

3.1. 概要

  • DockerでReactの環境を構築する際の手順

3.2. フォルダとファイルを作成する

mkdir app
touch Dockerfile
touch docker-compose.yml

3.3. Dockerfileを記述する

1FROM node:lts
2WORKDIR /usr/src/app

注釈

  • イメージはnodejsを選択する

3.4. docker-compose.ymlを記述する

 1version: "3"
 2services:
 3react:
 4   build:
 5      context: .
 6      dockerfile: Dockerfile
 7   tty: true
 8   environment:
 9      - NODE_ENV=development
10   volumes:
11     - ./app:/usr/src/app
12   command: sh -c "cd reactapp && yarn start"
13   ports:
14     - 3000:3000

注釈

  • serviceでreactコンテナを命名

  • environmentでnodejsの環境設定

  • commandでコンテナ起動時のコマンドを指定

  • volumesでローカルファイルをコンテナにマウントする

  • ポートはデフォルトの3000

3.5. コンテナのビルド

docker-compose build

3.6. Reactプロジェクトの作成

docker-compose run --rm react sh -c  "npx create-react-app react-app --template typescript"

3.7. コンテナの起動

docker-compose up -d

3.8. コンテナの終了

docker-compose down

3.9. 再ビルドを行う

docker-compose build
