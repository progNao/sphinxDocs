1. Railsの開発環境構築

1.1. 概要

  • UbuntuでRails7の環境を構築する際の手順

  • rbenvでruby環境を作成

  • gitはインストール済み

1.2. rbenvをインストール

sudo apt install rbenv

1.3. ruby-buildをインストール

git clone https://github.com/rbenv/ruby-build.git ~/.rbenv/plugins/ruby-build

1.4. rubyをインストール

rbenv install 3.x.x

警告

  • インストールには時間がかかる

1.5. インストールされたことを確認

ruby -v

1.6. rubyをグローバル化

rbenv global 3.x.x

1.7. bundleがインストールされていることを確認

bundle -v

1.8. railsをインストール

gem install rails

1.9. インストールされたことを確認

rails -v

1.10. railsアプリケーションを作成

rails new app
cd app

1.11. Gemfileを更新してgemをまとめてインストール

1gem "haml-rails"
2gem "sassc-rails"
3gem "font-awesome-rails"
4gem "devise"
5gem 'rails-i18n'

注釈

  • 不要な場合や、違う場合は別途自分で記述

1.12. 記述したgemをインストールする

bundle install
bundle update

1.13. railsを起動

rails s
